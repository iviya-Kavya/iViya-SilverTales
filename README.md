# Iviya — Silver Tales (Sample Site)

This is a small, responsive storytelling-style website built as a sample for the brand **Iviya – Silver Tales**.
It uses plain HTML/CSS/JS and is ready to be hosted on **GitHub Pages**.

## Files
- `index.html` — main page
- `styles.css` — styles
- `script.js` — small interactions
- `README.md` — this file

## How to publish on GitHub Pages (quick)
1. Create a new repository on GitHub (e.g. `iviyasilvertales.github.io` or any repo).
2. Upload the files from this package (or unzip and `git init` / commit / push).
3. In repository Settings → Pages, set branch to `main` (or `gh-pages`) and folder to `/root`.
4. Save — GitHub will publish at `https://<username>.github.io/<repo>` or `https://<username>.github.io/` if repo named `username.github.io`.

For step-by-step commands (if you prefer git):
```
git init
git add .
git commit -m "Initial Iviya — Silver Tales site"
git branch -M main
git remote add origin https://github.com/<your-username>/<repo>.git
git push -u origin main
```
